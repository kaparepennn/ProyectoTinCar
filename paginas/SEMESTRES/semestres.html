INFORMACIÓN SEMESTRES


Para la correcta gestion de este proyecto, se han trabajados diferentes secciones por semestre, que van ayudando poco a poco para que este proyecto pueda salir a flote correctamente y pueda ser funcional a futuro.

INFORMACIÓN PRIMER SEMESTRE


En este semestre desarrollamos actividades fundamentales para el inicio del proyecto, tales como el método Pico, modelo entidad relación, modelo relacional y por supuesto la base de datos para nuestro proyecto.

INFORMACIÓN SEGUNDO SEMESTRE


Para este segundo semestre trabajamos en varias cosas de acuerdo con lo propuesto en cada asignatura.

INFORMACIÓN TERCER SEMESTRE


Para este tercer semestre trabajamos en varias cosas de acuerdo con lo propuesto en cada asignatura, como lo fueron el documento Scrum para la asignatura de fundamentos de proyectos y el documento IEEE para la asignatura de ingeniería de software.
