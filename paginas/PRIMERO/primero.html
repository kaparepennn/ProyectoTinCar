PRIMER SEMESTRE


En este semestre desarrollamos actividades fundamentales para el inicio del proyecto, tales como el método Pico, modelo entidad relación, modelo relacional y por supuesto la base de datos para nuestro proyecto.

Modelo e implementación de bases de datos

Para esta asignatura trabajamos en el modelo relacional y modelo E-R (entidad relación) con los cuales nos guiamos para poder definir nuestra base de datos, la cual también creamos y hemos ido complementando con detalles necesarios y datos importantes.

Fundamentos de proyectos

Trabajamos en definir la estructura de trabajo de nuestro proyecto y el método pico.
